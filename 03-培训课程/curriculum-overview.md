# 课程体系总览

## 一句话结论

> 培训体系围绕HR六大核心模块设计，覆盖理论知识与专业技能双维度，配合模拟考试和真题解析，帮助学员系统备考、高效通关。

## 教学理念

| 理念 | 说明 |
|------|------|
| **以考促学** | 紧扣考试大纲，确保备考效率 |
| **理论+实操** | 不仅为了通过考试，更要掌握HR实战技能 |
| **系统化学习** | 六大模块循序渐进，构建完整知识体系 |
| **数字化升级** | 新增数字化HR、AI招聘等前沿内容 |

## 课程设计框架

```
                          培训课程体系
                               │
            ┌──────────┬───────┴───────┬──────────┐
            │          │               │          │
         四级课程     三级课程        二级课程    一级课程
         （入门）    （进阶）        （管理）    （战略）
            │          │               │          │
            └──────────┴───────┬───────┴──────────┘
                               │
                    ┌──────────┼──────────┐
                    │          │          │
                理论知识    专业技能    综合评审
                （机考）    （笔试）   （一/二级）
                    │          │          │
                    └──────────┴───────┬──┘
                               │
                    ┌──────────┼──────────┐
                    │          │          │
                HR六大模块  模拟考试   答疑辅导
```

## 六大核心模块

| 序号 | 模块 | 核心内容 | 考试比重 |
|------|------|---------|---------|
| 1 | 人力资源规划 | 组织架构设计、人员需求预测、供给预测 | 15% |
| 2 | 招聘与配置 | 招聘渠道、面试技术、人才测评 | 20% |
| 3 | 培训与开发 | 培训需求分析、培训体系设计、效果评估 | 15% |
| 4 | 绩效管理 | KPI设计、绩效考核方法、绩效反馈 | 20% |
| 5 | 薪酬福利管理 | 薪酬结构设计、薪酬核算、福利体系 | 15% |
| 6 | 劳动关系管理 | 劳动合同、社保、劳动争议处理 | 15% |

## 课程配套服务

| 服务项 | 说明 |
|--------|------|
| 官方教材 | 提供对应等级的官方教材 |
| 内部讲义 | 老师整理的重点归纳和考点解析 |
| 题库系统 | 在线刷题平台，含历年真题和模拟题 |
| 模拟考试 | 考前全真模拟，帮助熟悉考试节奏 |
| 答疑辅导 | 老师在线答疑，解决学习中的疑难问题 |
| 考试报名指导 | 协助完成报名流程和材料准备 |
