# 四级课程大纲

## 一句话结论

> 四级课程以HR基础知识为核心，侧重六大模块的基本概念和基础操作，适合零基础学员在2-3个月内完成系统学习。

## 核心信息速览

| 项目 | 内容 |
|------|------|
| 适合等级 | 四级（中级工/人力资源管理员） |
| 总课时 | 约80-120课时（以机构公布为准） |
| 课程周期 | 2-3个月 |
| 考试科目 | 理论知识 + 专业技能 |
| 教学方式 | 线下/线上/混合（以机构安排为准） |

## 课程模块与课时分配

| 模块 | 课时 | 内容重点 | 难度 |
|------|------|---------|------|
| **职业道德与基础知识** | 10课时 | 职业规范、组织行为学基础、企业文化 | 低 |
| **人力资源规划** | 10课时 | 组织结构类型、岗位分析基础、人员需求预测 | 低-中 |
| **招聘与配置** | 15课时 | 招聘渠道选择、简历筛选、面试基本方法 | 中 |
| **培训与开发** | 12课时 | 培训需求分析、培训方法、培训效果评估基础 | 低-中 |
| **绩效管理** | 15课时 | 绩效考核基本方法、KPI基础、绩效面谈 | 中 |
| **薪酬福利管理** | 12课时 | 薪酬基本概念、工资计算、社保公积金基础 | 中 |
| **劳动关系管理** | 12课时 | 劳动合同法基础、社保缴纳、离职管理 | 中 |
| **模拟考试与串讲** | 8课时 | 全真模拟考试、重点串讲、考前冲刺 | -- |

## 学习目标

完成四级课程后，学员应能够：

| 能力 | 具体表现 |
|------|---------|
| 基础认知 | 理解HR六大模块的基本概念和相互关系 |
| 操作能力 | 能独立完成基础人事操作（招聘发布、社保办理等） |
| 法规理解 | 熟悉劳动法律法规的基本条款 |
| 考试准备 | 掌握理论知识考试和专业技能考试的全部考点 |

## 常见问题

**Q: 四级课程需要每天花多少时间？**
A: 建议每天1-1.5小时，周末可适当增加。总备考周期2-3个月。

**Q: 四级课程和三级课程的差别大吗？**
A: 有差别但不大。四级侧重基础概念和基本操作，三级在此基础上增加了深入分析和方案设计能力。详见 [课程差异对比](./level-4-vs-3-comparison.md)。
