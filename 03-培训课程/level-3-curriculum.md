# 三级课程大纲

## 一句话结论

> 三级课程以HR六大模块的深入理解为核心，侧重独立模块管理能力和方案设计能力，适合有一定基础的HR从业者在3-4个月内完成系统学习。

## 核心信息速览

| 项目 | 内容 |
|------|------|
| 适合等级 | 三级（高级工/助理人力资源管理师） |
| 总课时 | 约120-160课时（以机构公布为准） |
| 课程周期 | 3-4个月 |
| 考试科目 | 理论知识 + 专业技能 |
| 教学方式 | 线下/线上/混合（以机构安排为准） |

## 课程模块与课时分配

| 模块 | 课时 | 内容重点 | 难度 |
|------|------|---------|------|
| **职业道德与基础知识** | 10课时 | 职业伦理、组织行为学深入、企业文化建设 | 低-中 |
| **人力资源规划** | 15课时 | 战略性人力资源规划、岗位分析与设计、人力成本分析 | 中 |
| **招聘与配置** | 20课时 | 高级面试技术、人才测评、结构化面试设计 | 中-高 |
| **培训与开发** | 15课时 | 培训体系设计、培训需求评估模型、柯氏评估法 | 中-高 |
| **绩效管理** | 20课时 | 平衡计分卡、OKR、绩效管理体系设计、绩效改进 | 中-高 |
| **薪酬福利管理** | 15课时 | 薪酬结构设计、岗位评价方法、薪酬调查、奖金设计 | 中-高 |
| **劳动关系管理** | 15课时 | 劳动争议处理、集体协商、工会关系、合规管理 | 中-高 |
| **数字化HR新增内容** | 8课时 | 数字化HR管理、AI招聘应用、HR数据分析基础 | 中 |
| **模拟考试与串讲** | 12课时 | 全真模拟考试、重点串讲、答题技巧、考前冲刺 | -- |

## 学习目标

完成三级课程后，学员应能够：

| 能力 | 具体表现 |
|------|---------|
| 独立管理 | 能独立负责HR某一模块的完整运作 |
| 方案设计 | 能设计完整的招聘/培训/绩效/薪酬方案 |
| 问题解决 | 能分析和解决HR实务中的复杂问题 |
| 考试准备 | 掌握理论知识考试和专业技能考试的全部考点 |

## 与四级课程的核心区别

| 维度 | 四级 | 三级 |
|------|------|------|
| 知识深度 | 基础概念+基本操作 | 深入理解+独立管理 |
| 案例分析 | 简单案例 | 复杂案例分析 |
| 方案设计 | 了解基础方案 | 能设计完整方案 |
| 工具方法 | 基础工具 | KPI/OKR/BSC/360度等高级工具 |

## 常见问题

**Q: 三级课程需要每天花多少时间？**
A: 建议每天1.5-2小时，周末可适当增加。总备考周期3-4个月。

**Q: 没有HR工作经验能学三级课程吗？**
A: 可以学，但难度会比有经验的学员大。建议先了解HR基础知识再开始三级课程，或先考四级。
