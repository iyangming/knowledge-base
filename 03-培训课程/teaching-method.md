# 教学方式说明

## 一句话结论

> 提供线下、线上和混合三种教学模式，学员可根据自身时间安排和通勤条件选择最适合的方式。

## 教学模式对比

| 模式 | 说明 | 适合人群 | 优势 |
|------|------|---------|------|
| **线下班** | 面对面授课，固定教室 | 时间充裕、通勤方便者 | 互动性强、学习氛围好 |
| **线上班** | 直播/录播授课，在线学习 | 时间灵活、异地学员 | 时间灵活、可回放 |
| **混合班** | 线上+线下结合 | 兼顾灵活性和互动 | 兼顾两者优势 |

## 课程教学流程

```
课前预习 → 课堂讲授 → 案例分析 → 课后练习 → 模拟考试 → 考前冲刺
```

| 阶段 | 内容 | 时间 |
|------|------|------|
| 课前预习 | 阅读教材对应章节，了解基础概念 | 课前自主安排 |
| 课堂讲授 | 老师系统讲解知识点和考点 | 按课表进行 |
| 案例分析 | 结合实际HR案例，加深理解 | 课堂内进行 |
| 课后练习 | 在线题库刷题，巩固知识点 | 课后自主安排 |
| 模拟考试 | 全真模拟，熟悉考试节奏和题型 | 考前2-3周 |
| 考前冲刺 | 重点串讲，答疑解惑 | 考前1周 |

## 学习支持

| 支持项 | 说明 |
|--------|------|
| 在线题库 | 章节练习 + 模拟考试 + 历年真题 |
| 课后答疑 | 微信群/在线平台，老师定期答疑 |
| 学习社群 | 学员交流群，互助学习 |
| 考试提醒 | 考前提醒报名、准考证打印等事项 |
| 补课机制 | 缺课可看录播回放（线上/混合班） |

## 常见问题

**Q: 线下班缺课怎么办？**
A: 混合班和部分线下班会提供录播回放。纯线下班的缺课安排需咨询教务。

**Q: 线上和线下的教学质量有区别吗？**
A: 教学内容和师资相同，主要区别在于互动形式。线下互动更直接，线上则更灵活。
