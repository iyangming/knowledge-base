# 补贴申领条件判断

## 一句话结论

> 申领补贴需同时满足三个核心条件：取得证书、累计缴纳失业保险满12个月以上、在获证后12个月内申请。

## 申领条件核对表

### 条件一：取得证书

| 要求 | 说明 |
|------|------|
| 证书类型 | 职业技能等级证书（企业人力资源管理师） |
| 证书状态 | 已颁发且可在 osta.org.cn 查询 |
| 证书等级 | 四级及以上 |

### 条件二：缴纳失业保险

| 要求 | 说明 |
|------|------|
| 累计缴纳 | 失业保险累计缴纳满 **12个月以上** |
| 缴纳状态 | 申请时失业保险处于正常缴纳状态（在职人员） |
| 缴纳地区 | 在申请补贴所在地区缴纳 |

> **重要**：不是所有在职人员都缴纳了失业保险。建议登录社保查询平台确认自己的失业保险缴纳情况。

### 条件三：申请时限

| 要求 | 说明 |
|------|------|
| 申请时限 | 证书颁发之日起 **12个月内** |
| 逾期后果 | 逾期不可补申请 |

## 三类申请通道

| 通道 | 适用人群 | 额外条件 |
|------|---------|---------|
| **在职人员通道** | 在职缴纳失业保险 | 累计缴满12个月 |
| **失业人员通道** | 处于失业状态 | 领取失业保险金期间或失业保险金领取期满 |
| **企业通道** | 企业统一申请 | 企业需满足职工岗位技能培训补贴条件 |

## 快速自测

```
你是否已取得企业人力资源管理师证书？
├── 否 → 先参加考试，取得证书后再申请
│
├── 是 → 是否累计缴纳失业保险满12个月？
│   ├── 不确定 → 登录社保平台查询确认
│   ├── 否 → 无法申请在职人员补贴（关注其他通道）
│   │
│   └── 是 → 是否在获证后12个月内？
│       ├── 是 → ✅ 满足条件，可以申请
│       └── 否 → ❌ 已超过申请时限
```

## 常见问题

**Q: 失业保险断缴过可以申请吗？**
A: 看累计缴纳是否满12个月。断缴不影响累计计算，但申请时需处于缴纳状态。

**Q: 自由职业者能申请吗？**
A: 如果以灵活就业身份缴纳了失业保险且满12个月，可以申请。否则需要通过其他通道。

**Q: 在A市考的证书，在B市申请补贴可以吗？**
A: 一般要求在社保缴纳地申请。跨市申请需符合当地政策。
