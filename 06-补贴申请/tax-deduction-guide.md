# 个税专项扣除指南

## 一句话结论

> 取得职业技能等级证书后，可在年度个税汇算清缴时申报继续教育专项扣除，额度为3600元/年。

## 政策依据

| 项目 | 内容 |
|------|------|
| 政策名称 | 个人所得税专项附加扣除（继续教育） |
| 扣除额度 | **3600元/年** |
| 适用证书 | 国家职业资格目录内的证书（含职业技能等级证书） |
| 申报时间 | 每年3-6月个税汇算清缴期间 |

## 申报步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 下载"个人所得税"APP | 国家税务总局官方APP |
| 2 | 进入"专项附加扣除填报" | 首页找到对应入口 |
| 3 | 选择"继续教育" | 选择扣除年度 |
| 4 | 填写证书信息 | 证书名称、证书编号、发证机关 |
| 5 | 提交申报 | 系统自动计算扣除额度 |

## 注意事项

| 事项 | 说明 |
|------|------|
| 申报时限 | 每年3月1日-6月30日 |
| 扣除方式 | 在预扣预缴或汇算清缴时扣除 |
| 证书要求 | 需在《国家职业资格目录》内 |

## 常见问题

**Q: 人力资源管理师证书在个税扣除目录内吗？**
A: 在。职业技能等级证书属于继续教育专项扣除范围。

**Q: 补贴收入需要交税吗？**
A: 技能提升补贴一般免个人所得税。
