# 补贴申请操作指南

## 一句话结论

> 通过"i深圳"APP或"广东人社"APP在线申请，提交证书信息后等待审核，约1-2个月到账。

## 申请方式一：i深圳APP（推荐深圳用户）

### 操作步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 下载"i深圳"APP | 应用商店搜索下载 |
| 2 | 注册/登录 | 使用深圳统一账号登录 |
| 3 | 进入"社保"板块 | 首页找到社保服务入口 |
| 4 | 找到"技能提升补贴" | 在社保服务中搜索 |
| 5 | 填写申请信息 | 输入姓名、身份证号、证书编号等 |
| 6 | 上传证书照片 | 拍照或上传证书扫描件 |
| 7 | 确认银行账户 | 补贴将打入该账户 |
| 8 | 提交申请 | 等待审核 |

## 申请方式二：广东人社APP

### 操作步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 下载"广东人社"APP | 应用商店搜索下载 |
| 2 | 注册/登录 | 使用广东省统一账号登录 |
| 3 | 进入"技能提升补贴" | 在服务列表中找到 |
| 4 | 填写并提交 | 同i深圳APP流程 |

## 审核与到账

| 环节 | 时间 | 说明 |
|------|------|------|
| 提交申请 | -- | 在线提交 |
| 系统初审 | 1-3个工作日 | 自动审核基本信息 |
| 人工复审 | 5-15个工作日 | 人社部门人工审核 |
| 公示 | 5-7天 | 部分地区需公示 |
| 拨付到账 | 1-2个月 | 打入指定银行账户 |

## 申请所需信息

| 信息 | 说明 |
|------|------|
| 个人身份信息 | 姓名、身份证号 |
| 证书信息 | 证书编号、证书等级、发证日期 |
| 联系方式 | 手机号 |
| 银行账户 | 用于接收补贴的银行卡号 |

## 注意事项

| 事项 | 说明 |
|------|------|
| 及时申请 | 获证后12个月内，逾期失效 |
| 信息准确 | 证书编号等信息需与证书一致 |
| 银行卡状态 | 确保银行卡处于正常使用状态 |
| 保留凭证 | 截图保存申请记录和审核进度 |

## 常见问题

**Q: 申请被退回了怎么办？**
A: 查看退回原因，修正后重新提交。常见原因：信息填写错误、证书信息不匹配等。

**Q: 审核多久能通过？**
A: 一般1-3周完成审核。如有疑问可拨打人社服务热线12333咨询。
