# HR从业者职业规划模板

## 一句话结论

> 职业规划需结合个人背景、兴趣和行业趋势。以下模板帮你梳理发展路径。

## 职业规划四步法

```
第1步：自我评估
├── 当前岗位和级别
├── HR相关知识掌握情况
├── 兴趣和擅长的HR模块
└── 长期职业目标

第2步：能力盘点
├── 已具备的HR技能
├── 需提升的能力短板
└── 计划考取的证书

第3步：路径设计
├── 短期目标（1-2年）
├── 中期目标（3-5年）
└── 长期目标（5-10年）

第4步：行动计划
├── 学习计划（证书+培训）
├── 实践计划（项目+轮岗）
└── 人脉建设（行业交流）
```

## 典型路径参考

| 起点 | 3年目标 | 5年目标 |
|------|---------|---------|
| HR助理 | HR专员+四级证书 | HR主管+三级证书 |
| HR专员 | HR主管+三级证书 | HR经理+二级证书 |
| HR主管 | HR经理+二级证书 | HRD+一级证书 |

## 能力发展重点

| HR发展阶段 | 核心能力 |
|------------|---------|
| 入门期（0-2年） | HR基础知识、人事操作、沟通能力 |
| 成长期（3-5年） | 模块专精、项目管理、数据分析 |
| 管理期（5-10年） | 团队管理、战略思维、业务理解 |
| 战略期（10年+） | 组织发展、变革管理、文化塑造 |
