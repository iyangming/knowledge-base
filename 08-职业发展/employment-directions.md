# 就业方向与岗位对照

## 一句话结论

> 持证后可从事HR全链条岗位，从HR助理到HRD，覆盖企业、政府、服务公司、教育机构等多类组织。

## 主要就业领域

| 领域 | 具体岗位 |
|------|---------|
| 企业HR部门 | HR专员、HRBP、HR经理、HRD |
| 政府及公共机构 | 人事处、组织部相关岗位 |
| 人力资源服务公司 | 猎头顾问、薪酬福利专员 |
| 教育培训机构 | 企业培训师、职业规划师 |
| 管理咨询公司 | HR咨询顾问 |

## 按证书等级对应岗位

| 等级 | 典型岗位 | 薪资范围（深圳参考） |
|------|---------|---------------------|
| 四级 | HR助理、人事专员 | 5-8K/月 |
| 三级 | HR专员、招聘/薪酬专员 | 8-15K/月 |
| 二级 | HR主管、HR经理 | 15-25K/月 |
| 一级 | HRD、CHO、VP | 25K+/月 |

## 新兴方向

| 方向 | 说明 |
|------|------|
| HRBP | 深入业务部门的HR伙伴 |
| 数字化HR | HR数据分析、HRIS管理 |
| 组织发展（OD） | 组织设计和人才发展 |
| 薪酬福利专家 | 专注薪酬设计和福利管理 |
