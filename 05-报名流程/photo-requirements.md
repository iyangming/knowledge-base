# 证件照要求

## 一句话结论

> 报名需提供符合规格的证件照（通常为近期白底免冠彩色照片），建议到专业照相馆拍摄。

## 照片规格

| 项目 | 要求 |
|------|------|
| 尺寸 | 通常为一寸或二寸（以评价机构要求为准） |
| 格式 | JPG/JPEG |
| 背景 | 白色 |
| 人物 | 免冠、正脸、露耳 |
| 时效 | 近6个月内拍摄 |

## 拍摄注意事项

| 注意事项 | 说明 |
|---------|------|
| 不佩戴眼镜 | 除非日常必须佩戴，否则建议摘下 |
| 不化浓妆 | 保持自然面貌 |
| 头发不遮挡 | 露出双耳和额头 |
| 着装建议 | 深色有领衣服，与白色背景区分 |

## 常见问题

**Q: 可以用手机拍照吗？**
A: 不建议。建议到专业照相馆拍摄，确保符合规格要求。

**Q: 可以用以前的照片吗？**
A: 需近6个月内拍摄的照片，过于陈旧的照片可能不被接受。
