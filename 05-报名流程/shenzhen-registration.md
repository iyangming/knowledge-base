# 深圳地区报名全流程

## 一句话结论

> 深圳报名分4步：联系备案评价机构领取报名码 → 凭码登录官方平台填报资料 → 审核通过后缴费 → 考前打印准考证。

## 官方报名平台

| 项目 | 内容 |
|------|------|
| **平台名称** | 广东省技能评价实名制平台 |
| **网址** | https://ggfw.hrss.gd.gov.cn/OUPX/#/skillLevelRecognition |
| **用途** | 广东省内技能评价统一报名平台 |

## 四步报名流程

### 第1步：联系备案评价机构，领取专属报名码

| 事项 | 说明 |
|------|------|
| 操作 | 联系在 pjjg.osta.org.cn 备案的深圳本地评价机构 |
| 获取 | 机构审核你的报考条件后，发放**专属报名码** |
| 前提 | 确认自己满足报考条件 → [报考条件速查表](../00-快速入门/eligibility-quick-check.md) |

> 注意：必须通过备案机构才能获取报名码，个人无法直接在平台注册报名。

### 第2步：凭报名码登录官方平台，填报资料

| 事项 | 说明 |
|------|------|
| 操作 | 在官方平台输入报名码，创建账号 |
| 填写内容 | 个人基本信息、学历信息、工作经历、报考等级等 |
| 上传材料 | 证件照、学历证明、工作证明等（详见材料清单） |
| 时间 | 在报名截止日期前完成 |

### 第3步：审核通过后缴费

| 事项 | 说明 |
|------|------|
| 审核 | 评价机构审核提交的资料 |
| 审核时间 | 一般3-5个工作日 |
| 缴费 | 审核通过后按通知缴费（考试费+培训费） |
| 确认 | 缴费成功后收到确认通知 |

### 第4步：考前一周打印准考证

| 事项 | 说明 |
|------|------|
| 打印时间 | 考前约一周 |
| 打印方式 | 登录官方平台下载打印 |
| 准考证信息 | 考试时间、地点、考场号、座位号 |
| 注意 | 准考证是考试当日入场的必须证件 |

## 报名流程图

```
联系备案评价机构
     ↓
审核报考条件 → 领取报名码
     ↓
登录官方平台（ggfw.hrss.gd.gov.cn）
     ↓
填写个人信息 + 上传材料
     ↓
等待审核（3-5个工作日）
     ↓
审核通过 → 缴费
     ↓
考前1周 → 打印准考证
     ↓
考试当日 → 携带身份证+准考证到考场
```

## 注意事项

| 事项 | 说明 |
|------|------|
| 报名截止 | 各考期报名截止时间不同，以官方通知为准 |
| 资料真实 | 所有报名信息必须真实，造假将取消资格 |
| 报名码唯一 | 每个报名码对应一次报考，请勿泄露 |
| 机构选择 | 务必确认机构已在 pjjg.osta.org.cn 备案 |

## 常见问题

**Q: 可以自己直接在平台报名吗？**
A: 不可以。必须先通过备案评价机构领取报名码才能在平台报名。

**Q: 报名码有效期多久？**
A: 一般在一个报名周期内有效，逾期需重新获取。

**Q: 审核不通过怎么办？**
A: 根据不通过原因补充材料后重新提交。如条件确实不满足，需等条件满足后再报名。
