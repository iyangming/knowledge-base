# 报名材料清单

## 一句话结论

> 报名需要准备4类核心材料：身份证明、学历证明、工作证明和证件照。建议提前2周准备齐全。

## 材料清单

### 必须材料

| 序号 | 材料 | 要求 | 备注 |
|------|------|------|------|
| 1 | **身份证** | 正反面复印件/扫描件 | 在有效期内 |
| 2 | **学历证明** | 毕业证书复印件/扫描件 | 或学信网学历认证报告 |
| 3 | **工作年限证明** | 原件（加盖公章） | 详见 [工作证明模板](../02-报考条件/work-experience-proof.md) |
| 4 | **证件照** | 电子版+纸质版 | 详见 [证件照要求](./photo-requirements.md) |

### 条件性材料（按需提供）

| 材料 | 何时需要 | 说明 |
|------|---------|------|
| 前置证书复印件 | 通过证书进阶通道报考时 | 四级/三级/二级证书复印件 |
| 职称证书复印件 | 通过职称通道报考时 | 初级/中级/高级职称证书 |
| 学信网认证报告 | 学历有疑问时 | 学历电子注册备案表 |

## 材料准备注意事项

| 注意事项 | 说明 |
|---------|------|
| **提前准备** | 建议报名开始前2周开始准备 |
| **真实性** | 所有材料必须真实有效 |
| **格式要求** | 电子材料一般要求JPG/PDF格式 |
| **清晰度** | 扫描件需清晰可辨认 |
| **盖章有效** | 工作证明需加盖单位公章 |

## 材料准备检查表

- [ ] 身份证正反面扫描件（清晰）
- [ ] 毕业证书扫描件（清晰）
- [ ] 工作年限证明原件（已盖章）
- [ ] 证件照电子版（符合规格）
- [ ] 前置证书/职称证书扫描件（如需要）
- [ ] 学信网学历认证报告（如需要）

## 常见问题

**Q: 学历证明丢了怎么办？**
A: 可以通过学信网（https://www.chsi.com.cn/）申请学历认证报告，或联系毕业学校补办毕业证明书。

**Q: 工作证明单位不愿盖章怎么办？**
A: 与单位HR部门沟通说明是职业技能等级考试报名要求。如实在无法盖章，咨询评价机构是否有其他替代方案。
