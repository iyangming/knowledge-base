# 报考条件速查表

## 一句话结论

> 报考条件只需记住一个规则：**满足任意一条即可报名**。以下速查表帮你根据学历+工作年限快速判断可以报考哪个等级。

## 四级（中级工）报考条件速查

**满足以下任意一项即可：**

| 条件通道 | 具体要求 | 典型人群 |
|---------|---------|---------|
| 工作经验通道 | 累计从事HR或相关工作满 **5年** | 有5年HR经验的从业者 |
| 证书+经验通道 | 取得五级/初级工证书后，累计工作满 **3年** | 已持初级证书者 |
| 学历通道 | 技工院校/中职/高职/**专科及以上**相关专业毕业（含应届生） | 大专/本科HR专业应届生 |

## 三级（高级工）报考条件速查

**满足以下任意一项即可：**

| 条件通道 | 具体要求 | 典型人群 |
|---------|---------|---------|
| 工作经验通道 | 累计从事HR或相关工作满 **10年** | 资深HR从业者 |
| 证书+经验通道 | 取得四级/中级工证书后，工作满 **4年** | 已持四级证书者 |
| 职称通道 | 取得 **初级职称** 后，工作满 **1年** | 有初级经济师/助理职称者 |
| 学历通道 | 高级工班及以上学历（含应届生），或持有四级证书+高校相关专业毕业 | 高学历HR从业者 |

## 二级（技师）报考条件速查

**满足以下任意一项即可：**

| 条件通道 | 具体要求 | 典型人群 |
|---------|---------|---------|
| 证书+经验通道 | 取得三级/高级工证书后，工作满 **5年** | 已持三级证书5年以上 |
| 职称通道一 | 初级职称 + 5年经验 + 取得三级证书满 **1年** | 有职称且有三级证书 |
| 职称通道二 | **中级职称** + 1年工作经验 | 有中级经济师职称 |
| 学历通道 | 高级技校/技师学院毕业生持三级证书 + 工作满 **2年** | 技师学院背景 |

## 一级（高级技师）报考条件速查

**满足以下任意一项即可：**

| 条件通道 | 具体要求 | 典型人群 |
|---------|---------|---------|
| 证书+经验通道 | 取得二级/技师证书后，工作满 **5年** | 已持二级证书5年以上 |
| 职称通道一 | 中级职称 + 5年经验 + 取得二级证书满 **1年** | 有职称且有二级证书 |
| 职称通道二 | **高级职称** + 1年工作经验 | 有高级经济师职称 |

## 快速自查流程图

```
你是否有HR相关工作经验？
├── 没有 / 刚毕业
│   ├── 大专及以上HR相关专业 → ✅ 可报四级
│   └── 非相关专业 → ❌ 需先积累工作经验
│
├── 有5年经验
│   └── ✅ 可报四级
│
├── 有10年经验
│   └── ✅ 可报三级
│
├── 已有四级证书
│   ├── 持证满4年 → ✅ 可报三级
│   └── 持证不满4年 → ❌ 继续等待
│
├── 已有初级职称
│   └── 取得后满1年 → ✅ 可报三级
│
└── 已有中级职称
    └── 取得后满1年 → ✅ 可报二级
```

## 关键概念说明

| 概念 | 说明 |
|------|------|
| **相关职业** | 企业管理、行政管理、管理咨询、管理研究等 |
| **相关专业** | 人力资源管理、劳动与社会保障、工商管理、企业管理等 |
| **工作年限证明** | 需加盖单位公章，注明"从事人力资源管理相关岗位" |
| **累计工作年限** | 不要求连续，累计即可 |

## 常见问题

**Q: 非HR专业能考吗？**
A: 能。相关专业只是学历通道之一，你也可以通过"工作经验通道"（累计5/10年相关工作经验）报考。

**Q: 工作经验必须是HR岗位吗？**
A: 不完全是。企业管理、行政管理、管理咨询等相关职业的工作经验也可计算。

**Q: 工作年限怎么证明？**
A: 需要单位开具工作证明，加盖公章，注明从事人力资源管理相关岗位及年限。

> 详见：[工作年限证明办理指南](../02-报考条件/work-experience-proof.md)

**Q: 可以跨级报考吗？比如直接考三级？**
A: 可以。只要有满足三级的任一通道条件，就可以直接报考三级，不需要先考四级。
