# 快速入门

## 一句话结论

> 企业人力资源管理师是HR领域最核心的职业能力证书，改革后效力不变、全国通用、可领补贴，深圳2026年三级补贴高达2600元。从了解到领补贴，本知识库帮你走完全流程。

## 核心信息速览

| 项目 | 内容 |
|------|------|
| 证书性质 | 职业技能等级证书（与原国家职业资格证书同等效力） |
| 证书等级 | 四级（中级工）→ 三级（高级工）→ 二级（技师）→ 一级（高级技师） |
| 发证机构 | 人社部备案的第三方评价机构（非政府直接发证，但效力等同） |
| 查询平台 | http://zscx.osta.org.cn（全国联网可查） |
| 深圳考试频次 | 每年2-4次（通常3月、7月、11月） |
| 深圳补贴标准 | 四级1500-2000元，三级**2600元**（2026年标准） |
| 考试科目 | 理论知识 + 专业技能（一/二级另加综合评审） |
| 备考周期 | 四级2-3个月，三级3-4个月 |

## 常见问题

**Q: 证书改革后还有用吗？**
A: 有用，效力完全不变。只是从"政府直接发证"改为"人社备案机构发证"，查询平台不变，全国通用，终身有效。详见 [政策改革脉络](../01-政策与证书/policy-reform-overview.md)。

**Q: 四级和三级选哪个？**
A: 大多数在深HR从业者建议直接报三级。如果零基础或工作年限不够，先考四级打基础。详见 [等级选择决策指南](./level-selection-guide.md)。

**Q: 考完能领多少钱补贴？**
A: 深圳地区2026年标准：四级1500-2000元，三级2600元。获证后1年内通过"i深圳"APP申请。详见 [深圳补贴标准](../06-补贴申请/shenzhen-subsidy-2026.md)。
