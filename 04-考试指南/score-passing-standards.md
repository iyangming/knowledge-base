# 成绩与合格标准

## 一句话结论

> 各科60分合格（百分制）。成绩一般在考后1-2个月公布。不合格的科目可在下次考试时补考。

## 合格标准

| 科目 | 满分 | 合格线 | 说明 |
|------|------|--------|------|
| 理论知识 | 100分 | 60分 | 单选+多选 |
| 专业技能 | 100分 | 60分 | 简答+计算+案例分析+方案设计 |
| 综合评审 | 100分 | 60分 | 论文+答辩（仅一/二级） |

## 成绩查询

| 项目 | 说明 |
|------|------|
| 查询时间 | 考后约1-2个月（以评价机构通知为准） |
| 查询方式 | 通过评价机构通知的渠道查询（官网/短信/APP） |
| 成绩构成 | 各科独立计分，需全部合格 |

## 补考规定

| 项目 | 说明 |
|------|------|
| 补考范围 | 仅限未合格的科目 |
| 补考费用 | 仅需缴纳补考科目的考试费 |
| 补考时间 | 下次考试周期（需重新报名补考科目） |
| 补考次数 | 一般不限制次数 |

## 成绩有效期

| 情况 | 说明 |
|------|------|
| 单科成绩保留 | 部分评价机构允许合格科目成绩保留一定期限（如1年），具体以机构规定为准 |
| 全科通过 | 考试成绩仅作为取得证书的依据，证书终身有效 |

## 常见问题

**Q: 一科没过，另一科的成绩保留吗？**
A: 取决于评价机构的规定。部分机构允许合格科目成绩保留1-2个考试周期，部分机构要求全部科目在一次考试周期内通过。具体咨询报名机构。

**Q: 成绩可以复核吗？**
A: 一般可以申请复核，但需在规定时间内提出。复核范围通常限于分数统计是否有误，不涉及主观题重新评判。
