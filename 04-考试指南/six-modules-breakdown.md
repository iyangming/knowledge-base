# 六大核心模块考点拆解

## 一句话结论

> 六大模块（规划、招聘、培训、绩效、薪酬、劳动关系）是考试核心，占专业技能科目72%的分值。以下按模块拆解高频考点。

## 模块一：人力资源规划

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 组织结构类型 | 高 | 直线制、职能制、矩阵制等的特点和适用场景 |
| 岗位分析 | 高 | 岗位说明书的编写方法 |
| 人员需求预测 | 中 | 定性预测法和定量预测法 |
| 人力成本分析 | 中 | 人工成本构成和计算方法 |

## 模块二：招聘与配置

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 招聘渠道 | 高 | 内部招聘vs外部渠道的优缺点 |
| 面试方法 | 高 | 结构化面试、行为面试、情景面试 |
| 人才测评 | 中 | 心理测验、评价中心技术 |
| 录用决策 | 中 | 录用原则和流程 |

## 模块三：培训与开发

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 培训需求分析 | 高 | 组织/任务/人员三层分析法 |
| 培训方法 | 高 | 在职培训、脱产培训、讲授法、案例法等 |
| 培训效果评估 | 高 | 柯氏四层评估模型（反应/学习/行为/结果） |
| 职业生涯规划 | 中 | 职业生涯发展阶段理论 |

## 模块四：绩效管理

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 绩效考核方法 | 高 | KPI、BSC、360度考核、MBO等 |
| 绩效指标设计 | 高 | SMART原则、指标权重设计 |
| 绩效面谈 | 中 | 面谈技巧、反馈方法 |
| 绩效改进 | 中 | 绩效差距分析方法 |

## 模块五：薪酬福利管理

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 薪酬结构 | 高 | 基本工资、绩效工资、奖金、津贴 |
| 岗位评价方法 | 高 | 岗位排序法、岗位分类法、要素计点法 |
| 薪酬调查 | 中 | 薪酬调查的目的和方法 |
| 薪酬计算 | 高 | 计时工资、计件工资、加班费核算 |
| 社保公积金 | 中 | 五险一金的缴纳比例和计算 |

## 模块六：劳动关系管理

| 考点 | 重要程度 | 说明 |
|------|---------|------|
| 劳动合同法 | **最高** | 劳动合同类型、期限、解除/终止条件 |
| 社会保险 | 高 | 养老、医疗、失业、工伤、生育保险 |
| 劳动争议处理 | 高 | 仲裁流程、调解、诉讼 |
| 工作时间与休假 | 中 | 标准工时、加班规定、带薪年假 |
| 民主管理 | 低-中 | 职工代表大会、工会 |

## 高频考点TOP 10

| 排名 | 考点 | 出现频率 |
|------|------|---------|
| 1 | 劳动合同法（签订/解除/经济补偿） | 极高 |
| 2 | 绩效考核方法（KPI/OKR） | 极高 |
| 3 | 培训需求分析方法 | 极高 |
| 4 | 薪酬结构设计 | 高 |
| 5 | 结构化面试技术 | 高 |
| 6 | 柯氏培训评估模型 | 高 |
| 7 | 岗位分析方法 | 高 |
| 8 | 劳动争议仲裁流程 | 高 |
| 9 | BSC平衡计分卡 | 中-高 |
| 10 | 招聘渠道选择 | 中-高 |
