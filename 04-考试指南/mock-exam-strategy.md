# 模拟考试策略

## 一句话结论

> 模拟考试是备考的关键环节，建议考前2-3周进行3-5次全真模拟，重点练习时间分配和答题节奏。

## 模拟考试的作用

| 作用 | 说明 |
|------|------|
| 熟悉题型 | 真实感受考试题型和难度 |
| 时间管理 | 练习在限定时间内完成答题 |
| 查漏补缺 | 发现薄弱环节，针对性复习 |
| 建立信心 | 通过模拟建立考试信心 |

## 模拟考试安排

| 阶段 | 时间 | 次数 | 目的 |
|------|------|------|------|
| 第一次模拟 | 考前3周 | 1次 | 了解考试全貌，发现薄弱点 |
| 第二次模拟 | 考前2周 | 1-2次 | 针对性改进 |
| 考前冲刺模拟 | 考前1周 | 1-2次 | 确认备考效果，调整状态 |

## 答题时间分配建议

### 理论知识（90分钟）

| 题型 | 题量 | 建议用时 |
|------|------|---------|
| 单选题 | 约80题 | 50分钟 |
| 多选题 | 约45题 | 30分钟 |
| 检查 | -- | 10分钟 |

### 专业技能（120分钟）

| 题型 | 建议用时 |
|------|---------|
| 简答题 | 25分钟 |
| 计算题 | 25分钟 |
| 综合分析题 | 35分钟 |
| 方案设计题 | 25分钟 |
| 检查 | 10分钟 |

## 模拟考试后的分析

每次模拟后，分析以下内容：

| 分析维度 | 具体内容 |
|---------|---------|
| 正确率 | 各模块得分情况，找出薄弱模块 |
| 时间分配 | 是否有时间不够的情况 |
| 错题类型 | 概念不清 / 审题不仔细 / 知识遗忘 |
| 改进方向 | 针对性制定改进计划 |
