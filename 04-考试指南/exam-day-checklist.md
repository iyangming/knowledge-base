# 考试当日清单

## 一句话结论

> 考试当天带齐证件（身份证+准考证），提前到达考场，保持冷静。以下是完整清单。

## 必带物品

| 物品 | 是否必须 | 说明 |
|------|---------|------|
| **身份证原件** | 必须 | 有效期内 |
| **准考证** | 必须 | 提前1周打印 |
| 黑色签字笔 | 必须 | 专业技能笔试用 |
| 2B铅笔 | 建议 | 理论知识机考一般不需要 |
| 橡皮 | 建议 | -- |
| 水 | 可选 | 透明瓶装水 |

## 考试当日流程

| 时间节点 | 事项 |
|---------|------|
| 考前30分钟 | 到达考场，找到考场教室 |
| 考前15分钟 | 身份验证，进入考场就座 |
| 考试开始 | 按指令开始答题 |
| 考试结束前15分钟 | 监考老师提醒时间 |
| 考试结束 | 停止答题，交卷离场 |

## 注意事项

| 事项 | 说明 |
|------|------|
| 考试地点确认 | 提前1天确认考场地址和交通路线 |
| 到达时间 | 建议提前30-40分钟到达，预留充足时间 |
| 休息充足 | 前一晚保证充足睡眠 |
| 饮食适度 | 考前吃适量食物，避免过饱或空腹 |
| 手机关机 | 考场内手机必须关机并放入指定位置 |

## 常见问题

**Q: 忘带身份证怎么办？**
A: 无法进入考场。务必前一晚准备好所有证件。

**Q: 迟到了怎么办？**
A: 开考后一般不允许入场。建议预留充足交通时间。

**Q: 考试可以去洗手间吗？**
A: 一般可以，但需举手示意由监考老师陪同。
