# 综合评审备考指南

## 一句话结论

> 综合评审仅二级和一级需要考，包含论文写作（约3000字）和现场答辩。建议提前3-4个月准备论文选题和写作。

## 适用等级

| 等级 | 是否需要综合评审 | 占总分比重 |
|------|-----------------|-----------|
| 四级 | 否 | -- |
| 三级 | 否 | -- |
| **二级** | **是** | 纳入总评 |
| **一级** | **是** | 纳入总评 |

## 论文写作要求

| 项目 | 要求 |
|------|------|
| 字数 | 约3000字 |
| 选题方向 | HR战略、组织发展、人才管理等 |
| 格式 | 标题、摘要、关键词、正文、参考文献 |
| 提交时间 | 考前提交（具体以评价机构通知为准） |

### 选题建议

| 方向 | 具体选题参考 |
|------|------------|
| 人力资源战略 | 数字化转型下的HR战略规划 |
| 绩效管理 | OKR在中小企业绩效管理中的应用 |
| 薪酬管理 | 弹性福利体系的设计与实施 |
| 人才发展 | 企业人才梯队建设策略研究 |
| 组织发展 | 扁平化组织结构下的人力资源管理 |

### 写作技巧

1. 理论+实践结合：引用HR理论框架+实际案例
2. 数据支撑：使用具体数据和图表
3. 对策可行：提出的方案应具有可操作性
4. 格式规范：严格遵守论文格式要求

## 答辩准备

### 答辩流程

| 环节 | 时间 | 内容 |
|------|------|------|
| 论文陈述 | 5-10分钟 | 概述论文核心观点和结论 |
| 老师提问 | 10-15分钟 | 围绕论文内容和HR专业知识提问 |
| 自由问答 | 5分钟 | 补充提问 |

### 答辩技巧

| 技巧 | 说明 |
|------|------|
| 熟悉自己的论文 | 对每个论点都能详细阐述 |
| 准备可能的问题 | 围绕论文内容预判提问方向 |
| 诚实作答 | 不确定的问题坦诚说明，不要编造 |
| 展示战略思维 | 体现HR管理层的全局观 |
| 控制时间 | 陈述时控制好节奏，不超时不遗漏 |

## 备考时间安排

| 阶段 | 时间 | 任务 |
|------|------|------|
| 选题与大纲 | 考前4个月 | 确定选题，完成论文大纲 |
| 论文初稿 | 考前3个月 | 完成论文初稿 |
| 修改完善 | 考前2个月 | 修改完善，定稿提交 |
| 答辩准备 | 考前1个月 | 准备答辩PPT，模拟练习 |

## 常见问题

**Q: 论文可以找人代写吗？**
A: 严禁代写。论文内容需与答辩时回答一致，代写被发现将取消成绩。且论文写作本身是学习和思考的过程。

**Q: 答辩很紧张怎么办？**
A: 多做模拟练习。自己对着镜子讲，或请同事朋友模拟提问。熟悉内容和流程后紧张感会大幅降低。
