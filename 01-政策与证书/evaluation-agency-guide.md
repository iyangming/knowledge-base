# 评价机构查询与选择指南

## 一句话结论

> 报名必须通过**人社部备案的评价机构**。报名前务必在 pjjg.osta.org.cn 核实机构资质，否则证书可能无效。

## 评价机构查询平台

| 项目 | 说明 |
|------|------|
| **平台名称** | 职业技能等级评价机构公示查询系统 |
| **网址** | https://pjjg.osta.org.cn/ |
| **用途** | 查询经人社部备案的评价机构 |
| **费用** | 免费 |

## 可查询内容

在 pjjg.osta.org.cn 可以查询到：

| 查询维度 | 说明 |
|---------|------|
| 评价机构名称 | 机构全称 |
| 备案机关 | 人社部或地方人社部门 |
| 评价职业/工种 | 机构被授权评价的职业和等级 |
| 机构类型 | 用人单位 / 社会培训评价组织 / 技工院校 |
| 备案状态 | 有效 / 已注销 |

## 机构类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| **用人单位** | 大型企业自主评价本单位员工 | 大型集团企业 |
| **社会培训评价组织** | 第三方专业评价机构（**学员最常见**） | 各地职业培训学校 |
| **技工院校** | 具备评价资质的职业院校 | 技师学院、技工学校 |

> 大多数个人考生通过"社会培训评价组织"报名，这类机构同时提供培训和评价服务。

## 如何选择评价机构（4步法）

### 第1步：核实备案资质

在 pjjg.osta.org.cn 搜索机构名称，确认：
- 机构已备案（状态为"有效"）
- 备案范围内包含"企业人力资源管理师"
- 备案等级覆盖你报考的等级

### 第2步：了解考试安排

| 了解事项 | 说明 |
|---------|------|
| 考试频次 | 每年组织几次考试 |
| 最近考期 | 下一期考试的报名和考试时间 |
| 考试地点 | 在哪里考试 |
| 考试形式 | 机考还是纸笔 |

### 第3步：对比服务内容

| 对比项 | 说明 |
|--------|------|
| 培训服务 | 是否提供培训课程、课时多少 |
| 教材资料 | 是否提供教材、题库 |
| 模拟考试 | 是否提供模拟考试系统 |
| 售后服务 | 考试未通过是否有重修政策 |

### 第4步：确认费用透明

| 费用类型 | 是否透明 |
|---------|---------|
| 考试费 | 应单独列明 |
| 培训费 | 应单独列明 |
| 教材费 | 应单独列明 |
| 其他费用 | 不应有隐性收费 |

> 详见：[费用明细](../05-报名流程/payment-info.md)

## 深圳地区机构选择

深圳市的评价机构可在 pjjg.osta.org.cn 筛选"广东省-企业人力资源管理师"查询最新备案名单。

**选择深圳本地机构的优势**：
- 考试地点在深圳，方便参加
- 培训课程针对深圳政策（如补贴政策）设计
- 后续服务（证书领取、补贴申请指导）更便利

## 常见问题

**Q: 不在备案机构报名会怎样？**
A: 在非备案机构报名取得的证书可能无法在 osta.org.cn 查询，相当于无效证书，也无法申请补贴。

**Q: 可以异地报名吗？**
A: 可以。评价机构的评价资质通常不限于本地。但实际考试需要到指定考点参加，异地报名意味着需要跨地区考试。

**Q: 多家机构都能报名，选哪家？**
A: 建议对比：培训质量、师资力量、考试安排、费用、通过率。选择性价比最高的正规备案机构。
