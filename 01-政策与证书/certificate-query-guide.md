# 证书查询操作指南

## 一句话结论

> 取得证书后，可通过人社部全国联网查询系统验证证书真伪。2025年2月起需输入姓名+证件号码+证书编号三项信息进行查询。

## 查询平台

| 项目 | 说明 |
|------|------|
| **平台名称** | 技能人才评价证书全国联网查询系统 |
| **网址** | http://zscx.osta.org.cn/ |
| **查询范围** | 职业技能等级证书 |
| **费用** | 免费 |
| **结果效力** | 官方可信，可用于企业验证 |

> 旧版国家职业资格证书查询地址：http://jndj.osta.org.cn/

## 查询步骤

### 步骤一：打开查询平台

浏览器访问 http://zscx.osta.org.cn/

### 步骤二：输入查询信息

**2025年2月12日起的查询要求**（三项必填）：

| 输入项 | 说明 | 示例 |
|--------|------|------|
| 姓名 | 证书上的姓名 | 张三 |
| 证件号码 | 身份证号码 | 4403XXXXXXXXXXXX |
| 证书编号 | 证书上的编号 | 见证书右下角 |

### 步骤三：查看查询结果

| 结果项目 | 说明 |
|---------|------|
| 姓名 | 部分隐藏显示（如张*） |
| 证件号码 | 部分隐藏显示 |
| 证书名称 | 企业人力资源管理师 |
| 证书等级 | 四级/三级/二级/一级 |
| 发证日期 | 证书颁发日期 |
| 评价机构名称 | 发证机构全称 |

> 注意：查询结果对姓名和证件号码进行了隐私保护（部分隐藏），这是正常的官方做法。

## 证书编号在哪找？

证书编号位于证书的**右下角**，格式通常为一串数字编码。如果找不到编号，可以联系发证的评价机构查询。

## 查询不到怎么办？

| 可能原因 | 解决方案 |
|---------|---------|
| 证书尚未录入系统 | 新发证书可能需要1-2周才能录入，耐心等待 |
| 输入信息有误 | 核对姓名、证件号码、证书编号是否正确 |
| 查询了错误平台 | 确认使用的是 zscx.osta.org.cn（技能等级证书）而非 jndj.osta.org.cn（旧版国考证书） |
| 机构未及时上报 | 联系发证评价机构确认是否已上报证书信息 |

## 常见问题

**Q: 查询结果和我的证书信息不完全一致怎么办？**
A: 如果是隐私保护导致的显示差异（如姓名隐藏），属正常现象。如果证书等级、发证机构等信息不一致，请联系评价机构核实。

**Q: 可以用手机查询吗？**
A: 可以。该网站支持手机浏览器访问，建议使用Chrome或系统自带浏览器。

**Q: 旧版国考证书怎么查？**
A: 如果你的证书是2020年之前通过国考取得的国家职业资格证书，请访问 http://jndj.osta.org.cn/ 查询。
