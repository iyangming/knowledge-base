# 相关专业认定清单

## 一句话结论

> "相关专业"和"相关职业"的范围比你想的更宽。除人力资源管理外，工商管理、企业管理、行政管理、劳动与社会保障等专业均可认定。不确定时以评价机构审核结果为准。

## 相关专业清单

### 核心相关专业（直接相关）

| 专业类别 | 具体专业 |
|---------|---------|
| 人力资源管理 | 人力资源管理、人力资源开发与管理 |
| 劳动经济 | 劳动与社会保障、劳动关系、劳动经济学 |
| 工商管理 | 工商管理、企业管理、工商企业管理 |
| 行政管理 | 行政管理、公共事业管理 |
| 管理学 | 管理学、管理科学 |

### 扩展相关专业（一般认可）

| 专业类别 | 具体专业 |
|---------|---------|
| 心理学类 | 应用心理学、人力资源管理心理学 |
| 社会学类 | 社会学、社会工作 |
| 法学类 | 法学（劳动法方向） |
| 经济学类 | 经济学、国民经济管理 |
| 会计/财务 | 会计学、财务管理（部分机构认可） |

### 技工院校相关专业

| 专业类别 | 说明 |
|---------|------|
| 人力资源管理 | 技工院校对应专业 |
| 工商管理类 | 技工院校对应专业 |
| 行政管理类 | 技工院校对应专业 |

> **重要提示**：以上清单基于常见认定情况，最终以报名时评价机构的审核为准。如有不确定的专业，建议提前咨询评价机构。

## 相关职业清单

报考条件中的"累计从事本职业或**相关职业**工作"包含以下职业：

| 职业类别 | 具体职业 |
|---------|---------|
| 人力资源 | 人力资源专员、招聘专员、薪酬专员、培训专员、HR助理 |
| 管理类 | 企业管理人员、行政管理人员、办公室管理人员 |
| 咨询类 | 管理咨询顾问、人力资源咨询顾问 |
| 研究类 | 管理研究人员、人力资源研究人员 |
| 培训类 | 企业培训师、职业培训师 |

## 如何确认自己的专业/职业是否被认可？

### 方法一：自行对照

对照上表，如果你的专业或职业在清单中，大概率可以认定。

### 方法二：咨询评价机构

报名前联系评价机构，说明自己的专业和职业背景，由机构确认是否满足条件。

### 方法三：提交审核

在报名时如实填写信息，由评价机构和官方平台审核。审核不通过不等于条件不满足，可能是材料不完整。

## 常见问题

**Q: 计算机专业能算相关专业吗？**
A: 一般不算核心相关专业。但如果你从事的是HR相关工作，可以通过工作经验通道报考。

**Q: 自考/成考学历可以用于学历通道吗？**
A: 可以。自考、成考、网络教育等国民教育系列学历均被认可，只要专业在相关范围内。

**Q: 港澳台/海外学历可以吗？**
A: 需要通过教育部学历认证后，方可使用。认证后的学历与国内同等学历享有同等待遇。

**Q: 工作经验中的"相关职业"怎么认定？**
A: 主要看你的岗位职责是否涉及HR相关工作（招聘、薪酬、培训、绩效、劳动关系等）。需要单位出具工作证明。详见 [工作年限证明办理指南](./work-experience-proof.md)。
