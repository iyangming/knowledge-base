# 工作年限证明办理指南

## 一句话结论

> 工作年限证明需由当前或曾就职单位开具，加盖公章，注明从事人力资源管理相关岗位及年限。造假将被取消证书并计入诚信档案。

## 证明要求

| 要求项目 | 具体说明 |
|---------|---------|
| **开具单位** | 当前或曾就职的用人单位 |
| **盖章要求** | 单位公章或人力资源部门章 |
| **内容要求** | 注明岗位名称、从事人力资源管理相关工作、工作年限 |
| **格式要求** | 一般无固定模板，但需包含必要信息 |

## 工作证明参考模板

```
工作年限证明

兹证明 _________（姓名），身份证号 _____________，
自 _____年__月至 _____年__月在我单位担任 _________（岗位名称），
从事人力资源管理相关工作，累计工作年限为 ___年___个月。

特此证明。

单位名称（盖章）：
单位地址：
联系人及电话：
日期：      年    月    日
```

## 注意事项

### 1. 岗位名称需体现HR相关性

推荐的岗位表述：
- 人力资源专员
- 人事专员
- 招聘专员
- 薪酬福利专员
- 培训专员
- 行政人事专员
- 人事主管
- HRBP

### 2. 工作年限计算规则

| 规则 | 说明 |
|------|------|
| 累计计算 | 不要求在单一单位连续工作，可累计多家单位的工作年限 |
| 截止时间 | 一般以报名当月为截止计算 |
| 兼职/实习 | 通常不计算在内（全日制在校期间的实习除外） |

### 3. 多家单位工作的情况

如果曾在多家单位从事HR相关工作：
- 每家单位分别开具证明
- 或由最后一家单位汇总开具（需能证明之前的工作经历）

### 4. 自由职业/个体经营

| 情况 | 处理方式 |
|------|---------|
| 自由职业HR顾问 | 可由客户单位出具项目工作证明 |
| 个体经营 | 提供营业执照等经营证明材料 |
| 劳务派遣 | 由实际用工单位开具证明 |

## 严禁造假

> **重要警告**：工作年限证明造假一经发现，将被取消考试资格或已取得的证书，并计入个人诚信档案。

- 不要虚报工作年限
- 不要伪造单位公章
- 不要虚构岗位经历
- 如实填写，审核不通过可补充材料

## 常见问题

**Q: 单位不愿意盖章怎么办？**
A: 建议与HR部门沟通说明这是职业技能等级考试的报名要求。如果实在无法盖章，可咨询评价机构是否有替代方案。

**Q: 已经离职了，原单位不配合怎么办？**
A: 可以尝试通过社保缴纳记录、劳动合同等辅助证明工作经历。具体需咨询评价机构。

**Q: 工作证明需要公证吗？**
A: 一般不需要。但个别评价机构可能要求额外材料，以机构要求为准。
